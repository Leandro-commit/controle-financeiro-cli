# 💰 Controle Financeiro CLI

Sistema simples em Python para controle de entradas, saídas e saldo.

## 🚀 Funcionalidades
- Adicionar entradas (ganhos)
- Adicionar saídas (gastos)
- Visualizar saldo atual
- Listar histórico de movimentações

## 🛠 Tecnologias
- Python

## ▶️ Como executar

```bash
python controle_financeiro.py
```
